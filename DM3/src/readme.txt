En lançant l'exécution à partir du fichier main, l'application s'ouvrira avec une fenêtre comportant 2 boutons.

Vous pourrez alors lancer une session Admin ou une session Client. Vous pourrez en ouvrir autant que désirez et 
naviguer dans l'interface. La majorité des fonctionnalités ne sont pas présentes de le GUI, mais vous pourrez
regarder les voyages disponibles en ce moment. Le Client affiche tous sans discrimination dans l'état actuel et
l'Admin peut sélectionner selon le type de voyage.
